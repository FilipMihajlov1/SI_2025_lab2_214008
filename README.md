## Цикломатска комплексност

Цикломатска комплексност се одредува по формулата V(G) = E - N + 2, каде Е е бројот на ребра во графот а N е бројот на јазли во графот.

V(G) = 24 - 19 + 2 = 7

Цикломатската Комплексност на графот е 7.
